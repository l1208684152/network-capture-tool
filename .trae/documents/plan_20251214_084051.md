# 电脑端网络抓包工具设计

## 功能需求

1. 列出当前运行的电脑进程，支持搜索过滤
2. 选择指定进程进行网络抓包
3. 实时显示抓包结果，包括源IP、目标IP、协议、端口、长度等信息
4. 支持开始/暂停/停止抓包
5. 支持保存抓包结果为pcap或CSV格式
6. 支持数据包详情查看

## 技术选型

* **UI框架**: tkinter (与现有项目保持一致)

* **抓包库**: pyshark (基于wireshark/tshark，功能强大，支持进程过滤)

* **进程管理**: psutil (获取运行中的进程列表)

* **数据处理**: pandas (处理和保存抓包数据)

## 主要模块设计

### 1. 进程选择模块

* 使用psutil获取当前运行的所有进程

* 提供进程搜索框，支持按进程名过滤

* 显示进程ID、进程名、CPU使用率等信息

* 支持单选或多选进程

### 2. 抓包控制模块

* 开始按钮：启动抓包线程

* 暂停按钮：暂停抓包

* 停止按钮：停止抓包并保存数据

* 抓包状态显示

### 3. 抓包结果显示模块

* 表格形式展示抓包结果

* 列包括：序号、时间、源IP、目标IP、协议、源端口、目标端口、长度、信息

* 支持点击查看数据包详情

* 支持实时更新

### 4. 数据保存模块

* 支持保存为pcap格式（可用于wireshark分析）

* 支持保存为CSV格式（便于Excel分析）

* 提供文件保存路径选择

## 关键代码设计

### 进程获取函数

```python
def get_running_processes():
    processes = []
    for proc in psutil.process_iter(['pid', 'name', 'cpu_percent', 'memory_percent']):
        try:
            processes.append({
                'pid': proc.info['pid'],
                'name': proc.info['name'],
                'cpu': proc.info['cpu_percent'],
                'memory': proc.info['memory_percent']
            })
        except (psutil.NoSuchProcess, psutil.AccessDenied):
            continue
    return sorted(processes, key=lambda x: x['name'])
```

```python
def start_capture(self, selected_processes):
    # 构建进程过滤条件
    pids = [str(p['pid']) for p in selected_processes]
    display_filter = f"tcp.port in (443, 80) or udp.port in (53)"  # 示例过滤
    
    # 使用pyshark开始抓包
    capture = pyshark.LiveCapture(
        interface='\Device\NPF_Loopback',  # 回环接口，可根据需要选择
        display_filter=display_filter
    )
    
    for packet in capture.sniff_continuously():
        if not self.running:
            break
        # 解析数据包并添加到结果列表
        self.parse_packet(packet)
```
